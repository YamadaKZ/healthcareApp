# ヘルスケアアプリケーション


## ヘルスケアアプリケーションとは
　AI英会話アプリケーションは、大規模言語モデルであるChatGPT APIと音声認識・音声合成の Web Speech APIを組み合わせて開発しています。またVOICEVOXを使った「ずんだもん」との会話機能やDictionary APIを使った単語検索機能も追加しています。機能としては、ユーザーがブラウザ上で質問を投げかけると、AIがそれに対して応答を生成しフィードバックを返してくれます。
